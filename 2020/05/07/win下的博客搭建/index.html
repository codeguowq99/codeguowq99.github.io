win下的博客搭建

教你如何在Windows下搭建自己的博客

背景声明

本文搭建博客的方法简单实用，不需要买服务器也不需要买域名，主要采用hexo框架在我们的Windows下搭建自己的博客框架

1、简介

Hexo是一个快速，简单且功能强大的博客框架。您使用Markdown（或其他标记语言）编写帖子，然后Hexo会在几秒钟内生成带有精美主题的静态文件。

2、前期准备

1、安装Node.js

（安装很简单，默认设置安装完成即可）

安装完成后在cmd下使用以下命令查看环境是否配置好了

npm -v

node -v

如果是这样就说明成功了

NPM是随同NodeJS一起安装的包管理工具，能解决NodeJS代码部署上的很多问题，常见的使用场景有以下几种：

  • 允许用户从NPM服务器下载别人编写的第三方包到本地使用。
  • 允许用户从NPM服务器下载并安装别人编写的命令行程序到本地使用。
  • 允许用户将自己编写的包或命令行程序上传到NPM服务器供别人使用。

2、安装Git

官网地址（https://git-scm.com/downloads）

Git是一个开源的分布式版本控制系统，可以有效、高速的处理从很小到非常大的项目版本管理。具体安装步骤如下：

下载Git：

点击下载位置进行下载

根据你的系统选择下载文件

搭建博客

1、安装cnpm

简介

淘宝团队做的国内镜像，因为npm的服务器位于国外可能会影响安装。淘宝镜像与官方同步频率目前为 10分钟 一次以保证尽量与官方服务同步。(就是cnpm比npm快)

安装

npm install -g cnpm --registry=http://registry.npm.taobao.org

查看cnpm版本

cnpm -v

2、下载hexo框架

下载

cnpm install -g hexo-cli

查看hexo框架版本

hexo -v

3、搭建博客

创建blog目录

必须要在根目录下面用下面的命令查看

chdir

创建blog目录

博客就是存放在这个目录下的，如果失败我们就把这个目录删掉从创建以下就好了

mkdir blog

进入blog目录

cd blog

在blog下生成博客

hexo init

启动博客

hexo s

会出现这写信息，我们复制红框里的信息到浏览器下就可以看到我们的博客生成了，停止用 Ctrl+C

这就是我们生成博客的样子

4、写博客

创建一篇博客

hexo s “my-first-blog"

使用vim打开文章

cd source/_posts

vim my-first-blog.md

写入内容

生成文章

cd ../.. #退回到blog目录下

hexo clean #清理

hexo g #生成

hexo s #执行

部署到网络

1、创建GitHub仓库

先登录到GitHub官网创建一各账号，然后在右上角加号哪里点击new repository新建一个仓库

在红框哪里输入仓库名字，必须和你的账号名字一样，我的也因为创建过了所以重复了，最后点击完成

2、在blog目录下部署一个git插件

cnpm install --save hexo-deployer-git

3、配置git环境变量

git config --global user.name "github用户名"

git config --global user.email "e-mail"

git config --global push.default simple

4、配置blog目录下的_config.yml文件

加入三行代码（注意冒号后必须有空格）

type: git

repo: https://github.com/codeguowq99/codeguowq99.github.io.git

branch: master

5、部署到远端

hexo d #部署到远端

有可能出现这个样子，这时需要输入用户名和e-mail

这样即部署成功

在浏览器中输入仓库名字即可看到博客

可能出现的错误

1、没有安装git，或者git安装失败，返回文章前面重新安装

2、博客的文章格式是markdown格式
